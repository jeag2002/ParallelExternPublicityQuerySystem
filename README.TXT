Java project created in Eclipse Oxygen Release (4.7) and Window 10

Java jdk1.8.0_101 (64 bits) (Mandatory jdk 1.8 64 version or superior)
Maven 3.3
JUnit 4.9

Create jar executing maven command:

mvn clean install 

(after all system generate a jar named ParallelExternPublicityQuerySystem-0.0.1-SNAPSHOT.jar)

Example of Execution (Windows 10 64Bits Version):

java -jar .\ParallelExternPublicityQuerySystem-0.0.1-SNAPSHOT.jar .\input\response.txt 3 10000
